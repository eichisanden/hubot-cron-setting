# hubot-setting

setting screen for hubot

See [`src/setting.coffee`](src/setting.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-setting --save`

Then add **hubot-setting** to your `external-scripts.json`:

```json
[
  "hubot-setting"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
